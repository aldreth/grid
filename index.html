Grid v0.1.0 Github Repo

A fluid grid, divided into equal parts which should be applied across all devices.

  • The grid is wrapped with 'skycom-container' which should be used to contain all your content, or individual modules.
  • The container is made up of up 12 columns 'skycom-1' through to 'skycom-12'. In Sass @include skycom(1) etc.
  • To float a column on the right, add the class 'right'.
  • To offset a column, add the class 'skycom-offset1' through to 'skycom-offset12'. In Sass @include skycom-offset(1) etc.
  • When the text shouldn't touch the edges of the columns use 'cushioned' class for padding. In Sass @include skycom-cushion(1) etc.

Grid 1140px

The default grid size is 1140px. You can limit this to 974px by adding the class grid-974 to the html tag.

 
 
 
 
 
 
 
 
 
 
 
 

lightbox Grid

The 'skycom-offset1' class is used to centre the lightbox. We then use another 'skycom-container' to create a new 12 column grid.
